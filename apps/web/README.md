# reenigne.dev

Next.js marketing site — landing, pricing, download, account, docs.

```bash
npm install
export NEXT_PUBLIC_API_URL=http://localhost:8000
npm run dev
```

Deploy to Vercel with domain `reenigne.dev`. Set:

- `NEXT_PUBLIC_API_URL=https://api.reenigne.dev`
- `NEXT_PUBLIC_RELEASES_URL=https://github.com/reenigne/reenigne/releases/latest`
